# MAX31
Чат-бот СГТУ "Волга бот"
Этот проект представляет собой чат-бота для мессенджера Max, который предоставляет информацию о Саратовском государственном техническом университете (СГТУ).

Предварительные требования

· Python 3.8 или выше

· pip (менеджер пакетов Python)

· Docker (опционально, для запуска в контейнере)

Локальный запуск
1. Установка зависимостей
```bash
!pip install aiomax
```
2. Настройка ключа доступа
Убедитесь, что файл key.py содержит корректный ключ доступа к API Max:
```python
key = "ваш_ключ_доступа"
```
3. Запуск бота
```bash
python max.py
```
Запуск через Docker
1. Сборка Docker-образа
Создайте файл Dockerfile в корневой директории проекта:
```dockerfile
FROM python:3.9-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .
CMD ["python", "max.py"]
```
Создайте файл requirements.txt:
```
aiomax
```
Соберите Docker-образ:
```bash
docker build -t sgtu-bot .
```
2. Запуск контейнера
```bash
docker run -d --name sgtu-bot-container sgtu-bot
```
Структура проекта
```
├── key.py          # Файл с ключом доступа
├── max.py          # Основной код бота
├── requirements.txt # Зависимости Python
└── Dockerfile      # Конфигурация Docker
```

Функциональность бота

· Главное меню: Основные разделы информации о вузе

· Внутренний мир вуза: Информация об институтах СГТУ

· Мероприятия вуза: Анонсы предстоящих событий

· Навигация: Возможность возврата в предыдущие меню

Логирование
Бот использует стандартное логирование Python. Уровень логирования установлен на INFO. Для изменения уровня логирования отредактируйте строку в коде:
```python
logging.basicConfig(level=logging.INFO)
```
Пример запуска через командную строку
```bash
# Активация виртуального окружения (рекомендуется)
python -m venv venv
source venv/bin/activate  # Linux/MacOS
# или
venv\Scripts\activate     # Windows
# Установка зависимостей
pip install aiomax
# Запуск бота
python max.py
```

Примечания

· Убедитесь, что ваш ключ доступа действителен и имеет необходимые разрешения

· Бот использует формат разметки Markdown для сообщений

· Для остановки бота используйте сочетание клавиш Ctrl+C в терминале

· При запуске в Docker убедитесь, что контейнер имеет доступ к интернету для работы с API
